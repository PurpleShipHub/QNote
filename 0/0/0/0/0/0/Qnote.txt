# Welcome to QNote! 📝

QNote is an open collaborative note-taking platform. Anyone can create and share notes using 6-digit room numbers.

## How it works:
1. **Enter a room number** (000000 to 999999) above
2. **Start writing** - your changes are saved automatically
3. **Share the room number** with others to collaborate
4. **All notes are public** and stored on GitHub

## Features:
- 🌍 **Open Source** - View the code on GitHub
- 🔄 **Real-time collaboration** - Share room numbers instantly  
- 📱 **Works everywhere** - No apps to install
- 🔗 **Direct links** - Share rooms via URL
- 💾 **Auto-save** - Never lose your work

## Try it out:
- **Room #123456** - Public sandbox for testing
- **Room #999999** - Community message board
- **Random room** - Click the shuffle button for a new room

## Examples:
- Meeting notes for team projects
- Shared shopping lists or todo items
- Collaborative writing and brainstorming
- Quick message exchange between devices
- Public documentation and wikis
